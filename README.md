## PDF Surgeon               

- PDF Surgeon is a GUI Application, use for extraction, combining pages from pdf files.
- Build with [python3 tkinter](https://docs.python.org/3.5/library/tkinter.html) and [PyPDF2](https://pythonhosted.org/PyPDF2/).
